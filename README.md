## 🚀 Deployment (CI/CD)

This project is deployed using **GitHub Actions + GitHub Pages**:

* Code pushed to `main` branch
* GitHub Actions workflow (`deploy.yml`) triggers automatically
* Site gets built and deployed to GitHub Pages
